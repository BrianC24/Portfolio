About Me

My interest in technology and gaming throughout my childhood has led me to web development. After creating my first web application it solidified my passion for this field. I would come across different classes online and teach myself how to code. With my continous interest I found Redwood Code Academy. Their intensive full-stack development program exposed me to multiple technologies. The fast paced program was something that I enjoyed. The ever-changing environment this field has, encompasses my desire to continously learn.

At Redwood Code Academy I learned various technologies to create web apps with CRUD functionalities. Working with a third pary API was something I grew to be comfortable with. I was able to implement my passion for gaming in my second project(Flag Rush). There I used JavaScript and Angular to be able to make pieces move on a board. Using SQL Server for my database I was able to store the state of the board so users can save their progress. Using technologies like CSS, Bootstrap, and SASS I found a passion for Web Design.

I worked on a project for R.M.F Photography to showcase his art and work as a photographer. You can see the rest of my projects as well as other technologies i know below.
